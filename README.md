# 📝 Simple Text Editor

## 📌 Overview
A **Python-based text editor** using `Tkinter`. Supports **basic file operations** like opening and saving `.txt` files.

## 🚀 Features
✔ **Open & Save Files**  
✔ **Simple GUI with Tkinter**  
✔ **Resizable Window**  
✔ **Word Wrapping for Text Entry**
